# Setup Instructions

## prerequisite
Have the following installed:
- Git
- Ruby
- Ruby on rails
## Clone and review it

Run the following commands to clone and run it.

### To clone the repository

`git clone git@github.com:Lichapa/budget-app-rails.git`

### To enter Directory

`cd budget-app-rails`

### To Install Gems

`bundle install`

### Run Tests

`rspec`

### running the program
- Open terminal and run `rails s`
